# Chest Cavity Legacy

**胸腔遗产** — 一个为 Minecraft 1.12.2 添加完整器官系统的模组。

> 版本: 1.0.0 | 作者: Shiver211 | 许可证: Apache License 2.0

---

## 简介

Chest Cavity Legacy 为 Minecraft 中的所有生物添加了一套功能性的**器官系统**。每个生物（玩家、动物、亡灵、傀儡、昆虫、末影人、龙等）都拥有一个内部"胸腔"——一个 27 格的物品栏，代表着它们的内脏器官。

玩家可以使用**开胸器**打开生物的胸腔，查看/替换器官，并通过器官评分系统改变实体的行为属性。

---

## 核心机制

### 胸腔系统

每个 `EntityLivingBase` 都会通过 Forge Capability 系统附加一个 `IChestCavity` 能力：

- **27 个器官槽位**：以 9×3 网格排列的物品栏
- **器官评分**：每个器官拥有多种属性评分，影响宿主的能力
- **器官兼容性**：器官带有所有者标记，不兼容的器官会导致**器官排斥**效果
- **数据驱动**：器官定义、胸腔类型、实体分配均通过 JSON 配置，支持用户自定义

### 器官获取

- 使用**剁刀**（6 个等级：木/石/金/铁/钻石/下界合金）击杀生物可获得器官
- **附魔影响**：
  - **外科手术** — 大幅提高器官掉落率
  - **医疗事故诅咒** — 使掉落的器官标记为不兼容
  - **手术恐惧症** — 完全阻止器官掉落
- 未打开胸腔的生物死亡时也会随机掉落器官（受抢夺附魔影响）

### 器官兼容性

- **无标记器官**：通用兼容（默认掉落）
- **O 型阴性附魔**：I 级使器官通用兼容，II 级还能在死亡时保留器官
- **所有者绑定器官**：仅与原始宿主兼容，移植到其他实体会触发器官排斥
- **排斥效果**：周期性伤害，可被禁用（配置选项）

## 器官评分系统

模组定义了 **47 种器官评分**，涵盖生存、战斗、探索、特殊能力等方面：

### 基础属性
| 评分 | 效果 |
|---|---|
| 生命值 | 增加最大生命值（每点 +4 HP） |
| 力量 | 增加攻击伤害 |
| 速度 | 增加移动速度 |
| 防御 | 减少受到的伤害（骨骼防御） |
| 神经 | 增加攻击速度和挖掘速度（急迫效果） |
| 幸运 | 增加幸运值 |
| 挖掘速度 | 增加挖掘速度 |

### 消化系统
| 评分 | 效果 |
|---|---|
| 消化 | 食物值倍率 |
| 营养 | 饱和度倍率 |
| 肉食消化/营养 | 肉类食物加成 |
| 草食消化/营养 | 植物类食物加成 |
| 腐烂消化 | 腐肉食物加成 |
| 铁胃 | 腐肉饱和度加成 + 移除饥饿效果 |
| 新陈代谢 | 消耗速率 |
| 耐力 | 减少消耗 |

### 呼吸系统
| 评分 | 效果 |
|---|---|
| 呼吸 | 呼吸能力 |
| 肺活量 | 水下空气容量 |
| 呼吸恢复 | 空气恢复速率 |
| 水下呼吸 | 水下呼吸能力 |

### 防御系统
| 评分 | 效果 |
|---|---|
| 火焰抗性 | 减少火焰伤害 |
| 冲击抗性 | 减少摔落/碰撞伤害 |
| 击退抗性 | 击退抗性 |
| 轻量化 | 减缓下落速度 |
| 跳跃 | 跳跃提升 + 减少摔落伤害 |

### 特殊能力（被动）
| 评分 | 效果 |

| 浮力 | 在空气/水中获得向上推力 |
| 发光 | 发光效果 |

| 恐水症 | 沾水时随机传送 |
| 诡异 | 苦力怕式自爆 |

### 战斗能力（被动）

| 凋零抗性 | 减少凋零效果持续时间 |

### 特殊能力（主动）
| 评分 | 效果 |
|---|---|

## 物品

### 工具
- **开胸器** — 打开生物胸腔的核心工具
- **剁刀**（6 级）— 用于屠宰生物获取器官的武器

### 器官类别

| 类别 | 示例 | 特点 |
|---|---|---|
| **人类器官** | 心脏、肺、肝、肾、胃、肠、脾、阑尾、肌肉、肋骨、脊柱 | 基础器官，可食用 |
| **腐烂器官** | 腐烂心脏、腐烂肺等 | 80% 几率附带饥饿效果 |
| **动物器官** | 动物心脏、动物肺等 | 来自普通动物 |
| **小型动物器官** | 小动物心脏、兔子心脏等 | 来自小型动物 |
| **特化肌肉** | 水生肌肉、鱼肉肌肉、野蛮肌肉、迅捷肌肉、弹性肌肉 | 不同运动特化 |
| **特化器官** | 鳃、羊驼肺、食肉动物胃、食草动物瘤胃等 | 特殊消化/呼吸 |
| **防火器官** | 防火心脏、防火肺等 | 来自火焰免疫生物 |
| **昆虫器官** | 昆虫心脏、丝腺、毒腺 | 有毒（食用附带中毒） |
| **末影器官** | 末影心脏、末影肺等 | 来自末影人 |
| **龙器官** | 龙心脏、龙肺、魔力反应器等 | 来自末影龙 |
| **烈焰构件** | 激活烈焰棒、烈焰外壳、烈焰核心 | 来自烈焰人 |
| **傀儡构件** | 傀儡电缆、傀儡装甲、傀儡核心 | 来自铁傀儡 |
| **其他特殊器官** | 苦力怕阑尾、潜影脾、气鳔、易爆胃等 | 各具特殊效果 |

### 食物加工

模组提供完整的肉类加工流水线：

```
器官/肉类 → 屠宰肉 → 生香肠 → 熟香肠
                  ↓
              丰盛香肠（使用更多材料）
                  ↓
              迷你香肠（小份版本）
```

- **香肠皮**：由肠类器官回收获得
- **有毒香肠**：使用昆虫器官制作
- **人肉香肠**：使用人类器官制作（有朊病毒风险）
- **外星/龙香肠**：使用末影/龙器官制作

## 附魔

| 附魔 | 稀有度 | 最高等级 | 效果 |
|---|---|---|---|
| **O 型阴性** | 稀有 | II | I 级：器官通用兼容；II 级：死亡时保留器官 |
| **外科手术** | 非常稀有 | III | 每级增加器官掉落率（+2 抢夺等效） |
| **医疗事故诅咒** | 非常稀有 | I | 诅咒。掉落的器官标记为不兼容 |
| **手术恐惧症** | 非常稀有 | I | 完全阻止器官掉落 |

## 快捷键

类别："Chest Cavity Organ Abilities"

| 快捷键 | 默认按键 | 功能 |
|---|---|---|
| **实用能力** | V | 循环切换：浮力、熔炉供能、铁修补、放牧、产丝 |
| **攻击能力** | R | 循环切换：自爆、龙息、龙弹、吐痰、喷火、恶魂、潜影弹 |

每种能力也有独立的按键绑定（默认无绑定）。

## 配置选项

配置文件位于 `config/chestcavity.cfg`，分为以下类别：

### 核心设置
- 默认胸腔类型
- 器官掉落率（抢夺加成、通用供体率）
- 器官排斥伤害和频率
- 心脏出血/肾脏中毒频率
- 各项器官评分对应的属性值
- 开胸器使用条件（生命值阈值）
- 是否允许打开其他玩家胸腔
- 死亡时是否保留器官
- 是否禁用器官排斥

### 能力调优
- 闪避距离、浮力提升、水晶合成范围/频率
- 防火/冲击防御系数、铁修补治疗百分比
- 光合作用频率、反刍参数、游泳速度因子
- 各项主动能力冷却时间

## 数据驱动系统

所有器官定义、胸腔类型和实体分配均通过 JSON 文件配置，支持用户自定义：

### 文件结构
```
assets/chestcavity/chestcavity_data/
├── organs/              # 器官定义（~100+ 文件）
│   ├── human/           # 人类器官
│   ├── animal/          # 动物器官
│   ├── small_animal/    # 小型动物器官
│   ├── rotten/          # 腐烂器官
│   ├── insect/          # 昆虫器官
│   ├── ender/           # 末影器官
│   ├── dragon/          # 龙器官
│   ├── blaze/           # 烈焰器官
│   ├── golem/           # 傀儡器官
│   ├── other/           # 其他特殊器官
│   ├── pseudoorgans/    # 伪器官（原版物品作为器官）
│   └── wither/          # 凋零相关器官
├── types/               # 胸腔类型定义（~50+ 文件）
│   ├── animals/         # 动物类型
│   ├── humanoids/       # 人形生物类型
│   ├── undead/          # 亡灵类型
│   ├── insects/         # 昆虫类型
│   ├── magical_beasts/  # 魔法生物类型
│   └── constructs/      # 构造体类型
└── entity_assignment/   # 实体到类型的映射（~30+ 文件）
```

### 自定义器官示例
```json
{
  "itemID": "minecraft:golden_apple",
  "organScores": [
    { "id": "chestcavity:health", "value": 2.0 },
    { "id": "chestcavity:luck", "value": 1.5 }
  ],
  "pseudoOrgan": true
}
```

用户可在 `config/chestcavity/data/` 目录下放置自定义 JSON 文件，会覆盖 assets 中的同名文件。

## 伪器官系统

原版物品也可作为器官使用（不会触发器官排斥）：

- **方块类**：泥土、熔炉、活塞、黑曜石、铁块、铁栏杆、荧石、骨块、金块、绿宝石块、钻石块
- **物品类**：骨头、腐肉、火药、烈焰棒、荧石粉、粘液球、TNT
- **食物类**：猪排、羊肉、牛肉
